¿QUE ES UN SISTEMA OPERATIVO?

Un sistema operativo es un conjunto de programas que permite manejar la memoria, disco, medios de almacenamiento de información y los diferentes periféricos o recursos de nuestro dispositivo

Un sistema operativo es un conjunto de programas que permite manejar la memoria, disco, medios de almacenamiento de información y los diferentes periféricos o recursos de nuestro dispositivo

Encontramos diferentes sistemas operativos como:

  • Microsoft Windows
  • MS-DOS
  • UNIX
  • MacOS
  • Ubuntu
  • Android

Dentro de las tareas que realiza el sistema operativo, en particular se ocupa de gestionar la memoria de nuestro sistema y la carga de los diferentes programas, para ello cada programa tiene una prioridady en función de la misma contará con los recursos de nuestro sistema por más tiempo que un programa de menor prioridad.
